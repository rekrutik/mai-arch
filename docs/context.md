# Контекст решения

```plantuml
@startuml
!include https://raw.githubusercontent.com/plantuml-stdlib/C4-PlantUML/master/C4_Container.puml

Person(admin, "Администратор")
Person(user, "Пользователь")

System(api, "Заказ услуг", "Сервис заказа услуг")

Rel(admin, api, "Просмотр, управление пользователями и услугами")
Rel(user, api, "Регистрация; создание, отправка заявок на приобретение услуги")

@enduml
```

## Назначение систем

|Система|Описание|
|-------|--------|
| Заказ услуг | Сервер, релизующий REST API, предоставляющий возможности по аутентификации пользователей, размещению услуг и заявок на них.
